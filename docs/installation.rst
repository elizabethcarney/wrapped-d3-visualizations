============
Installation
============


Stable release
--------------

To install Wrapped D3 Visualizations, run this command in your terminal:

.. code-block:: console

    $ pip install wrapped-d3-visualizations

This is the preferred method to install Wrapped D3 Visualizations, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Wrapped D3 Visualizations can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/elizabethcarney/wrapped-d3-visualizations

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/elizabethcarney/wrapped-d3-visualizations/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/elizabethcarney/wrapped-d3-visualizations
.. _tarball: https://github.com/elizabethcarney/wrapped-d3-visualizations/tarball/master
